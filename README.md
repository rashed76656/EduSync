## 🌟 Vision
**EduSync** is a high-fidelity institutional management platform designed to digitize and streamline the academic operations of **Rangpur Polytechnic Institute**. Built with a focus on **Premium Aesthetics** and **Administrative Efficiency**, it transforms manual tracking into a real-time, data-driven intelligence hub.

## ✨ Core Features

### 🏛️ Admin Command Center (V3)
- **Global Student Registry**: Centralized oversight for institution-wide student data with mass-promotion and alumni lifecycle management.
- **Financial Intelligence**: Real-time revenue analytics, collection charts, and a granular audit ledger for institutional fees.
- **Broadcast System**: Global administrative notification center for instant institutional announcements.
- **Institutional Identity**: Complete control over branding, logos, and official signage for academic documentation.

### 🍎 Teacher Workspace
- **Attendance Engine**: Rapid attendance marking with historical heatmaps and percentage tracking.
- **Academic Grading**: Comprehensive result entry system following the BTEB Diploma 2022 regulation.
- **Notice Board**: Departmental communication system for targeted student updates.

### 📄 Document Factory
- **Branded Transcripts**: On-demand PDF generation for student academic records with institutional branding.
- **Attendance Certificates**: Verified attendance summaries for academic audit.
- **Resilient Engine**: High-performance PDF generation with asynchronous error recovery for branding assets.

## 🎨 Design System
EduSync utilizes a **Premium Glassmorphism** aesthetic, featuring:
- **HSL Tailored Palettes**: Sophisticated rose and indigo gradients.
- **Backdrop Blurring**: High-fidelity translucent UI components for a modern SaaS feel.
- **Micro-animations**: Smooth transitions and hover effects using Tailwind CSS and Framer Motion logic.

## 🛠️ Technical Stack
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS.
- **State Management**: Zustand (Atomic State Architecture).
- **Backend / Auth**: Firebase Firestore & Firebase Auth.
- **PDF Generation**: jsPDF + AutoTable.
- **Visualization**: Recharts.
- **Deployment**: Vercel (CI/CD via GitHub).

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/rashed76656/EduSync.git
cd EduSync
```

### 2. Environment Configuration
Create a `.env` file in the root and add your Firebase and Cloudinary credentials (see `.env.example` for the template):
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_PROJECT_ID=your_project_id
# ... see .env.example
```

### 3. Install & Run
```bash
npm install
npm run dev
```
